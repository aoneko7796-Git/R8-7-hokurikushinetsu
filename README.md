# 7月16日実施　北陸信越運輸局 個人タクシー法令試験問題

GitHub Pages でそのまま公開できる PWA 版の練習アプリです。

## ファイル構成

- `index.html` … 公開用メインページ
- `style.css` … スタイル
- `script.js` … 問題データ・回答処理・採点ロジック
- `manifest.webmanifest` … PWA 設定
- `service-worker.js` … キャッシュ用サービスワーカー
- `動作確認用.html` … 1ファイルで確認できるローカル動作確認用

## 使い方

### GitHub Pages で公開する場合
1. このフォルダ内のファイルを GitHub リポジトリにアップロードします。
2. `Settings` → `Pages` で公開ブランチを設定します。
3. 数分待つと公開 URL が発行されます。

### ローカルで動作確認する場合
- `動作確認用.html` をブラウザで開いてください。

- 全40問
